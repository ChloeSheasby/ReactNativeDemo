# React Native Demo
This repo is for a demo React Native project for a Mobile Application Development course.

## What is React Native?
It uses native React development, a JavaScript-based library, to create user interfaces for iOS and Android.

## Pros and Cons of React Native
| **Pros**                             	| **Cons**                                            	|
|--------------------------------------	|-----------------------------------------------------	|
| Widely used                          	| Facebook supported                                  	|
| Community driven                     	| Difficult to update                                 	|
| Fast refresh                         	| Not efficient for complex applications              	|
| Expanding to Windows, macOS, and Web 	| Large companies like Intuit are moving away from it 	|
